# Programa de Inventario en Python
## Descripción

###### Este programa en Python permite registrar un producto dentro de un inventario simple.
###### El usuario ingresa el nombre del producto, su precio y la cantidad disponible.
###### El programa valida que los datos numéricos sean correctos y luego calcula el costo total del inventario multiplicando el precio por la cantidad.

# Funcionalidades

* ###### Solicita el nombre del producto.

* ###### Solicita el precio del producto con validación para evitar errores.

* ###### Solicita la cantidad del producto con validación para asegurar que sea un número entero.

* ###### Calcula el costo total del producto en el inventario.

* ###### Muestra todos los datos ingresados junto con el total.


# Cómo ejecutar el programa

* ###### Instalar Python en el sistema.

* ###### Guardar el archivo con el nombre inventario.py.

* ###### Abrir la terminal o consola.

* ###### Ejecutar el siguiente comando:

* ###### python inventario.py

* ###### Ingresar los datos solicitados por el programa.

## Ejemplo de ejecución
###### Ingrese el nombre del producto: Laptop
###### Ingrese el precio del producto: 1200
###### Ingrese la cantidad del producto: 3

# Resultado del inventario:
* ###### Producto: Laptop
* ######  Precio: 1200.0 
* ###### Cantidad: 3 
* ###### Total: 3600.0
